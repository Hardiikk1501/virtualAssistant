# 🤖 Voice-Enabled AI Assistant (MERN + Gemini AI)

An intelligent voice-enabled AI assistant built using the MERN Stack and Google Gemini AI. 
The assistant understands voice commands, engages in natural conversations, converts speech to text,
responds using AI-generated answers, and performs various real-world tasks through voice interaction.

🌐 Live Application: https://virtualassistant1-ox8r.onrender.com

## 📂 GitHub Repository

# 📖 Project Overview

This project is a voice-controlled AI assistant designed to provide a hands-free user experience. It combines Speech Recognition,
Natural Language Processing, and Text-to-Speech technologies to create a conversational AI system capable of understanding and responding to user commands in real time.

The assistant leverages Google Gemini AI to generate intelligent responses and can execute predefined actions such as opening websites,
searching YouTube, answering questions, and handling voice commands dynamically.

```text

        │
        ▼
Speech Recognition
        │
        ▼
Text Command
        │
        ▼
Command Processing
        │
 ┌──────┴────────┐
 │               │
 ▼               ▼
Custom Action   Gemini AI
 │               │
 └──────┬────────┘
        ▼

        │
        ▼
```

# 🔥 Key Functionalities

## Voice Command Workflow

1. User speaks a command
2. Speech Recognition converts speech to text
3. Command is analyzed
4. Assistant determines action
5. Gemini AI generates response if required
6. Response is converted to speech
7. User receives voice feedback

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/Hardiikk1501/virtualAssistant.git

cd virtualAssistant
```

## Backend Setup

```bash
cd backend

npm install

npm run dev
```

## Frontend Setup

```bash
cd frontend

npm install

npm run dev
```
